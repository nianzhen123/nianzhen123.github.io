log4j

  1. 创建append
    例如：创建基于控制台的append。
    log4j.appender.stdout=org.apache.log4j.ConsoleAppender
  2. 创建布局layout
    例如：log4j.appender.stdout.layout=org.apache.log4j.PatternLayout
  3. 说明使用什么用的格式
    例如：log4j.appender.stdout.layout.ConversionPattern=%n%-d{yyyy-MM-dd HH:mm:ss}%n[%p]-[Thread: %t]-[%C.%M()]: %m%n

  4. 说明什么情况下输出日志
    例如：log4j.rootLogger=debug, stdout, file
    意思是：第一个说明是日志级别，后面的就是你自己配置的append。
    rootLogger是根目录的logger,项目的classpath目录就是根目录

    log4j.logger.org.wang.controller = DEBUG,fout
    说明只会在org.wang.controller这个包中找
